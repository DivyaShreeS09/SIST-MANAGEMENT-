# SIST MANAGEMENT SYSTEM

A role-based approval system for managing:

- On Duty (OD) requests
- Lab Access requests
- Hostel Outpass requests

## Roles
Student  
Class Coordinator  
Year Coordinator  
HOD  
Mentor  
Chief Warden  
Warden  
Security

## Features
- Role-based login
- Multi-stage approval
- Request tracking
- Approval letter generation (PDF)
- Admin export reports

## Tech Stack
HTML  
CSS  
JavaScript  
LocalStorage (for demo data)
